# CV review rubric

Score each dimension **0–5** (5 = strong match). Provide 1–2 sentences of evidence each.

## Dimensions

1. **Must-have coverage** — Every explicit hard requirement in the JD addressed or honestly flagged as gap.
2. **Keyword / ATS** — Role titles, stack tokens, and synonyms appear naturally (no stuffing).
3. **Impact** — Bullets show outcomes, scope, metrics where credible.
4. **Market alignment** — Matches signals from `output/research/` (skills demand, title norms).
5. **Credibility** — No overclaim; timelines consistent; terminology fits seniority.
6. **Readability** — Concise bullets, consistent tense, no redundant buzzwords.

## Outputs required

- Table or list with scores + evidence
- **Top 5 prioritized edits** (most ROI first)
- **Risks**: gaps that could fail screening or interviews

## Enhancement handoff

- Each edit should map to YAML paths (`professional_experience[0].achievements`, etc.) when possible.
